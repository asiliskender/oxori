## Key Design Decisions (Locked)

1. **Optional Semantic Search Layer** — Entirely opt-in via `oxori embed`
2. **Binary Vector Storage** — `.oxori/vectors/*.vec` with IEEE 754 float32 (4–5× compact vs JSON)
3. **EmbeddingProvider Interface** — Pluggable, DI-based (no singleton)
4. **Staleness Detection** — SHA-256 content hash (not mtime)
5. **Cosine Similarity Scoring** — [0, 1] range, normalized vectors
6. **Stub Provider** — Hash-based, deterministic, offline testing
7. **OpenAI Integration** — Native Node.js `fetch()` only
8. **Error Codes** — 6 new codes defined for semantic layer
